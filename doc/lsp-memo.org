** html
html-mode

- vscode-html-language-server
- vscode-langservers-extracted

*** install
#+begin_src 
npm install -g vscode-langservers-extracted
#+end_src

*** 補足
HTMLファイルに直接記述するCSSやJavaScript
Eglot単体では1つのファイルに複数の言語が混ざる「Polyglot」な解析はやや苦手。
vscode-html-language-server を入れれば、ある程度のインラインCSS/JS補完は効く。
本格的にHTML内のJSも完璧に補完させたい場合は、LSPの追加ではなくEmacs側に polymode や web-mode という特殊なパッケージを導入し、ブロックごとにLSPを切り替えるハックが必要。

vscode-langservers-extractedをインストールすれば

- vscode-html-language-server (HTML用)
- vscode-css-language-server (CSS / SCSS / Less 用)
- vscode-json-language-server (JSON用)
- vscode-eslint-language-server (ESLint用)

をインストールしたのと同じ。のはず。。
